# Notebook
Created with MERN Stack
## Dependencies
- Express.js
- bcrypt
- express-validator
- jsonwebtoken
- MongoDB
- mongoose
- Node.js
- React.js
- react-router-dom
- font-awesome
  
## Installation
1. Clone the repo
```sh
git clone git@github.com:animeshmaiti/Notebook.git
```
2. Install NPM packages
```sh
npm install
```
